
SVM Image Retrieval System with Relevance Feedback

HOW TO RUN:

1. Install Python 3.8+
2. Install requirements:
   pip install -r requirements.txt

3. Put images inside dataset folder

4. Run:
   python main.py

CONTROLS:
Press 1 → Relevant
Press - → Irrelevant
Press any other key → Skip

The system improves retrieval precision using SVM feedback.
